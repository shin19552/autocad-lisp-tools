# autocad-lisp-tools

AutoCAD 用の AutoLISP ツール集です。

---

## ツール一覧

### BELT — ベルトラインコマンド (`BeltLine.lsp`)

2つの円を選択し、それらを囲むベルト状のポリラインを作成します。

**コマンド名:** `BELT`

**使い方:**

1. AutoCAD で `BELT` と入力してコマンドを起動
2. 1つ目の円をクリックして選択
3. 2つ目の円をクリックして選択
4. 2つの円を外接するベルト形状のポリラインが作成される

**エラー条件:**

| 状況 | メッセージ |
|------|-----------|
| 同一の円を2回選択した | 同じ円です。 |
| 円の中心が一致している | 中心点が同じため作成できません。 |
| 一方の円が他方を内包している | 円が近すぎるか、一方が他方を内包しているため作成できません。 |

**対応環境:**

- AutoCAD 2024 以下（LT 不可）
- ActiveX (VLA) が必要

---

## 使用方法（共通）

1. AutoCAD でコマンドラインに `APPLOAD` と入力
2. 使いたい `.lsp` ファイルを選択してロード
3. コマンドを入力して実行
